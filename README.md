# Nordic Life Navigator

AI-powered digital companion for international students and migrants in Sweden.

## Architecture

- **Frontend**: Next.js 14, Tailwind CSS, shadcn/ui
- **Backend**: Python 3.12, FastAPI, LangChain, Celery
- **Databases**: Firestore, Cloud SQL PostgreSQL, ChromaDB

## Commands

Run the backend locally:
```bash
make dev
```
